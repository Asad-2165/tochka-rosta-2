## 🚀 Как запустить на GitHub Pages (шаг за шагом)

### Шаг 1 — Создай аккаунт на GitHub
Зайди на [github.com](https://github.com) и зарегистрируйся (если ещё нет).

### Шаг 2 — Создай новый репозиторий
1. Нажми кнопку **«New»** или **«+»** → **«New repository»**
2. Название репозитория: `dqi` (или любое другое)
3. Поставь галочку **«Public»** (обязательно для бесплатного GitHub Pages)
4. Нажми **«Create repository»**

### Шаг 3 — Загрузи файлы
**Вариант A — через браузер (без Git):**
1. Открой созданный репозиторий
2. Нажми **«uploading an existing file»** или **«Add file» → «Upload files»**
3. Перетащи ВСЕ файлы из папки `dqi/`:
   - `index.html`
   - `app.html`
   - `manifest.json`
   - `sw.js`
   - `icon.svg`
4. Напиши комментарий: `Initial commit`
5. Нажми **«Commit changes»**

**Вариант B — через Git (командная строка):**
```bash
git init
git add .
git commit -m "Initial DQI commit"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГИН/dqi.git
git push -u origin main
```

### Шаг 4 — Включи GitHub Pages
1. Зайди в репозиторий → **Settings** (Настройки)
2. В левом меню найди раздел **«Pages»**
3. В разделе **«Source»** выбери:
   - Branch: **`main`**
   - Folder: **`/ (root)`**
4. Нажми **«Save»**
5. Подожди **1–2 минуты**

### Шаг 5 — Открой сайт
После включения GitHub Pages появится ссылка вида:
```
https://ВАШ_ЛОГИН.github.io/dqi/
```

Сайт готов! Приложение доступно по адресу:
```
https://ВАШ_ЛОГИН.github.io/dqi/app.html
```

## 📱 Как установить приложение на телефон

### iPhone / iPad (iOS):
1. Открой `https://ВАШ_ЛОГИН.github.io/dqi/app.html` в **Safari**
2. Нажми кнопку **«Поделиться»** (квадрат со стрелкой вверх) внизу экрана
3. Прокрути вниз и выбери **«На экран "Домой"»**
4. Нажми **«Добавить»**
5. Приложение появится на главном экране как нативное!

### Android (Chrome):
1. Открой `https://ВАШ_ЛОГИН.github.io/dqi/app.html` в **Google Chrome**
2. Нажми на **три точки** (⋮) в правом верхнем углу
3. Выбери **«Установить приложение»** или **«Добавить на главный экран»**
4. Нажми **«Установить»**
5. Приложение установлено!

> 💡 Chrome также может автоматически показать баннер «Установить DQI App» через 3 секунды после открытия.

## 🛠️ Если что-то не работает

**Service Worker не регистрируется:**
Это нормально при открытии файлов через `file://`. Нужен HTTPS — т.е. именно GitHub Pages.

**Шрифты не загружаются:**
Нужно подключение к интернету. При первом открытии шрифты кешируются.

**PWA-баннер на iOS не появляется:**
iOS не показывает автоматический баннер — нужно вручную: Safari → Поделиться → На экран Домой.
